## Task 3: Wire attempt-aware escalation into ClaimTask iteration check

**Files:**
- `internal/ops/claim_task.go` — replace direct iteration comparison with classifyLimitEscalation dispatch
- `internal/ops/claim_task_test.go` — add 2 tests

**Changes:**

1. **`internal/ops/claim_task.go`** — Replace the iteration limit check block (lines 151-163):

   Current code:
   ```go
   if strategy.enforceIterationLimit() && task.Iteration >= maxCoderIterations {
       blockedIteration, blockedLimit, err := enforceRejectedIterationLimit(bb, taskID, agentID, taskStatus, pipelineTransitions)
       ...
   }
   ```

   New code:
   ```go
   if strategy.enforceIterationLimit() {
       reviewLimit := effectiveReviewCycleLimit(state.Config)
       escalation, shouldEscalate := classifyLimitEscalation(
           task.ReviewCyclesCurrent,
           reviewLimit,
           task.Iteration,
           maxCoderIterations,
           task.EffectiveAttempt(),
       )
       if shouldEscalate {
           switch escalation.action {
           case LimitActionNewAttempt:
               result, taErr := TransitionToNewAttempt(projectRoot, taskID, escalation.reason)
               if taErr != nil {
                   return nil, fmt.Errorf("failed to transition to new attempt: %w", taErr)
               }
               return nil, &PreconditionError{Reason: fmt.Sprintf(
                   "task %s exhausted limits, transitioned to attempt %d — claimable on next cycle",
                   taskID, result.NewAttempt,
               )}
           case LimitActionBlocked:
               blockedIteration, blockedLimit, err := enforceRejectedIterationLimit(bb, taskID, agentID, taskStatus, pipelineTransitions)
               if err != nil {
                   return nil, fmt.Errorf("failed to enforce iteration limit: %w", err)
               }
               return nil, &PreconditionError{Reason: fmt.Sprintf(
                   "task %s reached max iterations (%d/%d) and was transitioned to BLOCKED",
                   taskID, blockedIteration, blockedLimit,
               )}
           }
       }
   }
   ```

   Note: `state.Config` is already available from line 72 (`readTaskState`) and used at line 135 (`effectiveCoderIterationLimit`). `TransitionToNewAttempt` is called before Phase 2 (worktree handling), so no worktree work is wasted.

2. **`internal/ops/claim_task_test.go`** — Add 2 tests:
   - `TestClaimTask_IterationCapAttempt1_TriggersNewAttempt`: REJECTED task at attempt 1 with `Iteration == maxCoderIterations`. ClaimTask returns PreconditionError containing "transitioned to attempt". Task in state has `Attempt == 2`, initial pipeline status, `Iteration == 0`, `ReviewCyclesCurrent == 0`.
   - `TestClaimTask_IterationCapAttempt2_TriggersBlocked`: REJECTED task at attempt 2 with `Iteration == maxCoderIterations`. ClaimTask returns PreconditionError containing "transitioned to BLOCKED". Task in BLOCKED status.

**done_when:** (a) The iteration limit check in `ClaimTask` (around line 151) uses `classifyLimitEscalation` with `task.EffectiveAttempt()` instead of direct `task.Iteration >= maxCoderIterations` comparison. (b) `LimitActionNewAttempt` calls `TransitionToNewAttempt` and returns PreconditionError to abort the claim. (c) `LimitActionBlocked` delegates to existing `enforceRejectedIterationLimit()` (preserving its atomic re-check). (d) `TestClaimTask_IterationCapAttempt1_TriggersNewAttempt` passes: REJECTED task at attempt 1 hitting iteration limit → PreconditionError returned, task transitioned to initial status with Attempt=2. (e) `TestClaimTask_IterationCapAttempt2_TriggersBlocked` passes: REJECTED task at attempt 2 hitting iteration limit → PreconditionError returned, task in BLOCKED.

   ```go
   func ensureRejectedWorktreeExists(
       gitWrapper *git.Git,
       ctx *claimContext,
   ) (claimWorktreePhaseResult, error) {
       result := claimWorktreePhaseResult{}
       branchName := paths.TaskBranchPrefix + ctx.taskID

       if _, err := os.Stat(ctx.worktreeDir); err == nil {
           // Worktree exists — validate and preserve
           if err := validateRejectedWorktree(gitWrapper, ctx.taskID, ctx.worktreeDir, ctx.worktreeRel); err != nil {
               return result, err
           }
           return result, nil
       }

       // Worktree missing — recreate from integration branch (recovery path)
       _ = gitWrapper.DeleteBranch(branchName) // cleanup stale branch if any
       if _, err := gitWrapper.CreateWorktree(ctx.taskID, ctx.integrationBranch); err != nil {
           return result, fmt.Errorf("failed to recreate worktree for REJECTED task from integration branch: %w", err)
       }
       result.created = true
       return result, nil
   }
   ```
