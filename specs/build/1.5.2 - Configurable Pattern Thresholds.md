# Configurable Pattern Thresholds

## User Story

As a maintainer, I want configurable pattern thresholds so circuit-breaker sensitivity can be tuned without code changes.

## Acceptance Criteria

- [ ] Pattern trigger thresholds can be configured per pattern type.
- [ ] Configuration changes are validated before affecting runtime behavior.
- [ ] Threshold configuration is traceable and auditable.
- [ ] Default thresholds preserve current detection behavior when no overrides are set.
