# Automatic Rescope Prompts

## User Story

As a planner, I want automatic rescope prompts when iteration/review limits are exhausted.

## Acceptance Criteria

- [ ] Exceeded iteration or review-cycle limits trigger explicit planner prompts.
- [ ] Prompts include enough task context to support immediate rescoping decisions.
- [ ] Prompting behavior avoids duplicate or noisy escalations.
- [ ] Rescope outcomes are tracked as explicit lifecycle decisions.
