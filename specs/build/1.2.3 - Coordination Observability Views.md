# Coordination Observability Views

## User Story

As a maintainer, I want coordination observability views so role-state drift is visible quickly.

## Acceptance Criteria

- [ ] Coordination views expose planner/coder/reviewer status and current assignments in a single place.
- [ ] Observability highlights stale leases, blocked queues, and claim starvation signals.
- [ ] Views remain consistent with blackboard source-of-truth state.
- [ ] Operators can identify coordination regressions without inspecting raw YAML manually.
