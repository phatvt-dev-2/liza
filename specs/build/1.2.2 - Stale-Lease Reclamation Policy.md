# Stale-Lease Reclamation Policy

## User Story

As an operator, I want explicit stale-lease reclamation policy so abandoned claims do not block throughput.

## Acceptance Criteria

- [ ] Stale coder and reviewer leases can be detected deterministically.
- [ ] Reclamation transitions preserve task integrity while returning claimable work to the queue.
- [ ] Reclaimed tasks and lease resets are auditable in shared state history.
- [ ] Reclamation policy distinguishes safe auto-reclaim from cases requiring planner intervention.
