# Mandatory High-Risk Skill Enforcement

## User Story

As a collaborator, I want mandatory high-risk skills (debugging/testing/review) so critical work follows a repeatable methodology.

## Acceptance Criteria

- [ ] High-risk triggers force required skill loading before execution.
- [ ] Missing or inaccessible required skills produce explicit blocked/error behavior.
- [ ] Enforcement applies uniformly across pairing and multi-agent execution paths.
- [ ] Compliance is auditable through prompts/logged workflow artifacts.
